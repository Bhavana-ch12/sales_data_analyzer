# Sales Data Analyzer (Python)

## 📌 Description
This project analyzes sales data using Python and generates useful insights.

## 🚀 Features
- Calculates total sales
- Finds top-selling product
- Shows monthly sales trends
- Displays results using graphs

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

## ▶️ How to Run
1. Install libraries:
   pip install pandas matplotlib

2. Run the program:
   python main.py
